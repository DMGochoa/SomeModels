# RegresionLineal

Para un conjunto de puntos, encontrar la recta $y=Ax+b$ que mejor se acomode a esa nube de puntos.

## Preguntas para usar regresion lineal

- Tengo que predecir sobre una variable numerica?
- Las variables independientes con las que cuento son primordialmente numericas?
- No cuento con una gran cantidad de variables y/o variables categoricas con muchos niveles.

## Recomendaciones

Si tienes que predecir variables numericas empieza con regresion lineal, si no funciona, salta a otros modelos. Reduce las variables lo mas que puedas. Cuidado con la multicolinealidad! No predigas fuera del dominio de la variable independiente.
